A reimagination of the classic Snake Game built with Python's turtle graphics library, featuring 3 difficulty levels!
Built as a fun exploration of game development and interactive programming

FEATURES:

1.Classic Snake Game gameplay

2.Three difficulty modes: Easy, Normal and Hard.

3.High score boards for each difficulty level.

4.Custom sound effects using winsound.

SKILLS I USED:

1.Object Oriented Programming

2.Collision detection

3.Data Persistence

4.Working with multiple game modes

5.Importing custom sounds using winsound
